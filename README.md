## Webpack

### Tutorial:

- 首先你需要全局安装 webpack: `npm install -g webpack`



- 首先你也学会使用 ES6
- 然后你要学会使用 react
- 然后你要学会使用 webpack
    - 你要学会配置基本的 webpack.config.js
    - 然后你要学会使用 loader 和 plugins
    - 然后你还要学会 webpack-dev-server
    - 把命令添加进 `npm scripts` 配置里
- 然后你要学会使用 babel
- 最后你还要学会把 webpack 和 babel 还有 babel 结合在一起
